Testimonial Plugin for ImpressPages CMS
Compatible with version 2.4+


HOW TO INSTALL
===============
1. Upload this plugin "testimonial" folder to "ip_plugins/comment/" directory of your website.
2. Login to administration area
3. Go to "Developer -> Modules" and press "install".
4. Refresh admin area and click on "Community" in admin tab, you will see Testimonial is now available to use.
5. Testimonial widget should also be available now in the widget bar, drag and drop Testimonial widget to any page on your website you want Testimonial to display.


HOW TO USE
===========

Open Community / Testimonial tab and add some testimonials.
Open Standard / Content Management tab and drag testimonials widget on the page.
